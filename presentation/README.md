# PharmGuard AI – Presentation assets

Place short screen-recording GIFs or videos here for the demo:

1. **auto_approve.gif** – Streamlit: user asks for Aspirin 75mg → auto_approve, order created.
2. **prescription_required.gif** – User asks for Azithromycin → require_prescription message; then (optional) Create order with prescription URL.
3. **refill_alert.gif** – Admin panel: Refill alerts for user u100 (or another user with history).

## How to record

- **Windows**: Use Xbox Game Bar (Win+G) or OBS to record the Streamlit window; convert to GIF with a tool like ScreenToGif or ezgif.com.
- **macOS**: QuickTime screen recording, then convert to GIF (e.g. ffmpeg or online converter).
- **Linux**: SimpleScreenRecorder or OBS; ffmpeg to convert to GIF.

Suggested length: 10–30 seconds per clip.
